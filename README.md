
# CCNA Packet Tracer Labs

This repository contains 34 hands-on networking labs built using Cisco Packet Tracer during my CCNA preparation.

## Topics Covered
- VLAN Configuration
- Inter-VLAN Routing
- Static Routing
- RIP and OSPF
- DHCP Configuration
- NAT
- Access Control Lists (ACL)
- Subnetting and VLSM

These labs helped me practice real network configurations and troubleshooting in simulated enterprise environments.
